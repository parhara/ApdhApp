POSTING UMKM

1.     Isi jaminan utama “WIRAUSAHA”

 

 

2.     Pilih kode Jaminan “11. WIRASWASTA /UMKN” dan di jaminan tambahan diketik “KARTU KELUARGA”

 

 

3.     Jenis  jaminan Pilih kode “11. WIRASWASTA /UMKN” dan pastikan dikotak jenis jaminan sudah “11. WIRASWASTA /UMKN”

 

4.     Pada bagian jamin ini isi seperti berikut / “KOSONGKAN”

 

5.     SIMPAN DATA

6.     Jika data sudah disimpan silahkan, sebelum mencetak perjanjian kredit posting dulu jaminan klik tombol “Input Jaminan Lain2” seperti gambar berikut.

 

7.     Setelah muncul menu input jaminan silahkan panggil nomor kredit atau cari nama nasabah seperti gambar dibawah

 

8.     Kemudian ketik di jenis jaminan tambahan dan simpan satu pertsatu

 

9.     Setelah disimpan pastikan hasilnya muncul disebelah kanan

Dan lanjutkan posting persyaratan lain “bukuk nikah, ijazah dll”

 

10.                         Setelah selesai silahkan keluar dari menu jaminan tambahan.

 

11.                        Silahkan cetak Perjanjian Kredit

 

12.                        Berikut hasi Output perjanjian
